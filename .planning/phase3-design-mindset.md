# Phase 3 設計思維整理（依你口述）

▋ 我對你設計思維的理解

1. **按鈕 = 整張卡片，不要小圓圈**
   - 三個付款方式：只顯示「很小的圖示（icon）+ 標籤」，**不要**前面再多一個選單用的小圓圈（radio 視覺）。
   - 點擊時：**整張卡片**就是按鈕，點卡片任意處即選取，不需要再去點一個圓圈。
   - 選中態：整張卡片有邊框/光暈即可（如參考圖 Card、Carte bancaire）。

2. **下方內容與上方一體、不割裂**
   - 信用卡卡號、有效期限、安全碼的表格要與上方按鈕區**視覺一致**（間距、邊框、圓角、對齊），不要卡號前面一大段空白、也不要大小不一。
   - 只顯示「目前選到的那一種付款方式」對應的內容：選信用卡 → 只出現信用卡表單；選 ATM → 只出現 ATM 一句說明；選超商 → 只出現超商一句說明。

3. **訊息只講一次、放對位置**
   - **最下方**只保留**一段**總說明：「使用 PayUNi（統一金流）付款。信用卡可站內刷卡並進行 3D 驗證，ATM/超商將直接取號顯示於收據頁。」不再在區塊上方或中間重複同樣意思。
   - 信用卡區塊內：只保留一條「安全加密付款，由統一金流處理，卡號不會儲存於本站。」**刪掉**「請放心，我們將由統一金流加密你的個人資料」這段重複文案。
   - ATM/超商：只顯示該方式一句說明（例如「送出後將於收據頁顯示轉帳帳號與期限」），其餘都靠最下方那一段總說明即可。

4. **互動簡單**
   - 第一步就是選付款方式；選了哪一個，下面就只出現那一個的內容（連動）。
   - 點第二個選項（例如 ATM 轉帳）時，**直接點那張卡片**就切換，不需要再點小圓圈。
